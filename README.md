# 🛒 E-Commerce App - Backend

This is a simple and secure **RESTful API** backend for an E-Commerce web application, built with **Node.js**, **Express.js**, and **MongoDB**. It includes user authentication, product management, order handling, and image uploads using **Cloudinary**.

---

## 🚀 Features

- 🔐 **User Authentication**
  - JWT-based login and registration
  - Role-based access control (Admin/User)

- 📦 **Product Management**
  - CRUD operations for products
  - Image upload with Cloudinary
  - Category-wise filtering

- 🛒 **Cart & Orders**
  - Add/remove/update cart items
  - Place and manage orders

- 🧑‍💼 **Admin Features**
  - Manage users, products, and orders

- 📤 **File Upload**
  - Multer for local handling
  - Cloudinary for cloud storage

---

## 🧰 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB** with **Mongoose**
- **JWT** for authentication
- **Multer** for file uploads
- **Cloudinary** for media hosting
- **dotenv**, **CORS**, **Helmet**, etc.
